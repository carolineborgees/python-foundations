# 🐍 Python Foundations

Repositório de aprendizado progressivo em Python, do zero ao avançado com IA aplicada.

Criado para documentar minha jornada de transição para Engenharia de Dados e ML/AI Engineering —
e para ajudar outros devs que estão começando o mesmo caminho.
